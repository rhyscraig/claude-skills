# Security Policy

## Reporting a Vulnerability

**Do not** open a public GitHub issue for security vulnerabilities.

Instead, please email: **security@craighoad.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We will acknowledge receipt within 24 hours and provide updates on our investigation.

## Security Practices

- All commits are signed
- Secrets are never committed (checked by CI)
- Dependencies are regularly updated
- Code is reviewed before merge to main
