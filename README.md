# Claude Skills

Production-grade platform for building Claude skills with reusable patterns, guardrails, configuration management, and comprehensive documentation.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)

## Quick Start

```bash
pip install claude-skills
```

## Features

- **Configuration Hierarchy** — Master config + per-repo overrides
- **Safety Guardrails** — Confirmation gates, audit logging, rate limiting
- **Clean Architecture** — Layered design with separated concerns
- **Type Safety** — Type hints and validation throughout
- **Best Practices** — Production-grade error handling and documentation
